# 🚀 AI Resume Analyser (Frontend)

A modern web application that analyzes resumes using AI and provides insights to improve them. Built with cutting-edge frontend technologies for a smooth and responsive user experience.

---

## 📌 Features

- 📄 Upload resume files
- 🤖 AI-powered resume analysis
- 📊 Detailed feedback and insights
- ⚡ Fast and responsive UI
- 🌙 Dark/Light theme toggle
- 🔗 API integration with backend services

---

## 🛠️ Tech Stack

- **Frontend:** React 19, TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Routing:** TanStack Router
- **Animations:** Framer Motion
- **HTTP Client:** Axios

---

## 📂 Project Structure
src/
