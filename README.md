# 🚀 AI Resume Analyser (Frontend)

A modern web application that analyzes resumes using AI and provides insights to improve them. Built with a fast and responsive frontend stack.

---

## 📌 Features

- 📄 **Upload resume files** - 🤖 **AI-powered resume analysis** - 📊 **Detailed feedback and insights** - ⚡ **Fast and responsive UI** - 🌙 **Dark/Light mode support** - 🔗 **Backend API integration** ---

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript  
- **Build Tool:** Vite  
- **Styling:** Tailwind CSS  
- **Routing:** TanStack Router  
- **HTTP Client:** Axios  
- **Animations:** Framer Motion
